# The Cocktail DB with React

App usando api rest y de fuentes múltiples de bebidas y cócteles de todo el mundo.

# Api Reference

https://www.thecocktaildb.com/

# Demo 

https://sensational-pithivier-e34aee.netlify.app

## Install

```
npm install
```

## Run for development

```
npm run start
```

## Build to production

```
npm run build
```
